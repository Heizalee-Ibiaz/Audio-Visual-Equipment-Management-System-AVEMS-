# Audio-Visual-Equipment-Management-System-AVEMS-
AVEMS is an application for students, teachers, professors, and both academic and non-academic organizations of San Sebastian College â€“ Recoletos de Cavite to conveniently book and reserve audio-visual (AV) equipment needed for various activities or events.
